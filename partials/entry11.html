Unit Testing - Front End

April 3, 2017

For the front end, we used jasmine with protractortest to do end to end UI testing.

This allows us to check if clicking on each of the location buttons led to the appropriate page. Similarly, this tested that sign up and log in buttons also works.

Although at this stage of our app, this testing isn’t as valuable, once our other features are up , it will be able to efficiently test the UI.

Commands to run the tests:
webdriver-manager start (to start up the server)
protractor conf.js (to run the test)
