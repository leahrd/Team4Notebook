February 2, 2017

Description

Our project will be a campus nutrition web app for Tufts University.

Features

  • See current day menu options by dining hall
  • View nutritional and allergy warnings for each menu item
  • Maintain a food diary
  • Rate meals and save favorites
  • Receive optional notifications when favorite offerings are available
  • Auto check-in to dining halls and see which other users are also at the same dining hall
